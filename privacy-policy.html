隐私政策

最后更新日期：2026年1月29日

重要提示：我们非常重视您的隐私。本隐私政策说明了"Orange 聊天记账"应用如何收集、使用和保护您的个人信息。

1. 信息收集

1.1 您主动提供的信息

在使用"Orange 聊天记账"应用时，您可能会提供以下信息：

  • 交易记录：包括交易类型（收入/支出）、金额、描述、分类和日期
  • API Key：用于访问硅基流动 AI 服务的密钥，安全存储在您设备的 Keychain 中

1.2 自动收集的信息

我们不会自动收集您的任何个人身份信息、设备信息或使用数据。

2. 信息使用

2.1 核心功能

  • AI 智能分类：您输入的自然语言记账内容会发送到硅基流动 AI API 进行分析和分类
  • 数据存储：您的交易记录存储在设备本地的 SwiftData 数据库中
  • iCloud 同步：如果您启用了 iCloud，交易记录会自动同步到您的 iCloud 私有数据库，仅您可以访问

2.2 数据处理流程

当您使用聊天记账功能时：

  1. 您输入的文本发送到硅基流动 API（使用 DeepSeek-V3.2 模型）
  2. AI 分析文本并返回交易类型、金额、分类等信息
  3. 解析后的交易记录保存到本地数据库
  4. 如果启用 iCloud，数据自动同步到您的私有 CloudKit 数据库

3. 第三方服务

3.1 硅基流动 AI API

  • 服务商：硅基流动（SiliconFlow）
  • 用途：自然语言理解和交易信息提取
  • 数据传输：仅传输您输入的记账文本内容
  • 隐私政策：https://siliconflow.cn

重要说明：硅基流动 API 仅接收您输入的文本内容，不会接收您的个人身份信息、设备信息或其他敏感数据。

3.2 Apple iCloud

  • 服务商：Apple Inc.
  • 用途：同步您的交易记录到多个设备
  • 数据存储：私有 CloudKit 数据库，仅您可以访问
  • 隐私政策：Apple 隐私政策

4. 数据安全

4.1 安全措施

  • API Key 加密：您的硅基流动 API Key 使用 iOS Keychain 加密存储，只有在设备解锁时才能访问
  • HTTPS 传输：所有网络通信使用 HTTPS 加密
  • 本地存储：交易记录存储在设备本地，受 iOS 沙盒保护
  • iCloud 加密：iCloud 数据传输和存储均由 Apple 加密

4.2 数据访问控制

  • 应用不会将您的数据分享给任何第三方（除了提供服务所必需的硅基流动 API）
  • 应用不会将您的数据用于广告或营销目的
  • 应用不会跟踪您的行为或收集分析数据

5. 数据保留和删除

5.1 数据保留

  • 本地数据：永久保留在您的设备上，直到您手动删除
  • iCloud 数据：永久保留在您的 iCloud 账户中，直到您手动删除

5.2 数据删除

您可以随时：

  • 在应用内删除单条交易记录
  • 在设置中清除 API Key
  • 删除应用以清除所有本地数据
  • 在 iCloud 设置中关闭应用的 iCloud 同步，并删除 iCloud 数据

6. 儿童隐私

本应用不针对 13 岁以下的儿童。我们不会有意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，发现您的孩子向我们提供了个人信息，请联系我们，我们将及时删除。

7. 您的权利

您对自己的数据拥有完全的控制权：

  • 访问权：您可以随时在应用中查看所有交易记录
  • 修改权：您可以编辑任何交易记录
  • 删除权：您可以删除任何交易记录或清除所有数据
  • 导出权：您的数据存储在标准的 iOS 数据库中，可以通过 iCloud 备份导出

8. 隐私政策变更

我们可能会不时更新本隐私政策。更新后的隐私政策将在本页面发布，并更新"最后更新日期"。如果有重大变更，我们会在应用内通知您。

9. 适用法律

本隐私政策受中华人民共和国法律管辖。如果您不在中国大陆地区，当地法律可能适用。
